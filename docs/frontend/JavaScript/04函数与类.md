### 函数

> 函数是 `JavaScript` 中的一等公民

每个函数都有属性 `__proto__`、`prototype` 

~~~javascript
// 普通函数
function a() {}
// 构造函数
function b() {
  this.name = 'aha'
}

// 原型一样
a.__proto === b.__proto__ // true
// this. 指向构造函数的实例
new b().name // 'aha'
// 在构造函数的原型上定义的属性，实例可以通过原型链访问
b.prototype.age = 10
new b().age // 10
~~~

### 类

js 中的类也是基于函数的实现

~~~javascript
class Test {
  // 实例的属性
  a = 1
  constructor() {
  	// 实例的属性
    this.b = 1
  }
  // 在原型上定义的属性，实例可以通过原型链访问
	c() {
    console.log('c')
  }
  // 实例的属性
	d = function() {
    console.log('d')
  }
}
~~~
